# 标语集

## 主 slogan（首选，用于页面大标题 / 分享卡片）

- **你是《人民的名义》里的哪一号？**
- **汉东省给你建了一份档案**
- **32 题，在 16 位干部里照见自己**

## 落地页副标题候选

- 脱胎于剧中四条政治哲学光谱，在 16 位剧中人物里找到最像你的那一位
- 本测评不套 MBTI 模板，对应的是剧里真正吵架的那些分歧
- 5 分钟，看看你是侯亮平、李达康，还是那个太想进步的祁同伟

## 短 tagline（140 字内用，微博 / 朋友圈开场）

- 【干部人格档案】你是哪一号？32 题，5 分钟
- 终于有人做了《人民的名义》版 MBTI
- 测了下我是 ______（测完再甩结果截图）
- 不是 INTJ，是沙瑞金。不是 ENFP，是侯亮平
- 16 位汉东干部在等你，你最像谁？

## 极短（Twitter/X / 推送标题）

- 人民的名义 · 人格测评
- 汉东省人格档案室
- 你是哪一号干部？
- In Their Name · Persona Test（英文备选，国际用户）

## 按钮微文案

- 开始评估（主 CTA）
- 开始测评（备选）
- 立即上报（再备选，更公文）

## 结果分享文案（用户点"复制链接"/"生成长图"时弹窗里展示）

- "同志，我的档案已归档。你来测一下？"
- "我测出来是 XXX，你呢？"
- "汉东省给我立了一份档案，扫码看——"

## 印章类微文案（用于视觉元素）

- 绝 · 密（首页）
- 已 · 阅（引语下方）
- 已归档（结果页长图右下）
- 诚实填报（答题页底部提示可选）

## 避免使用的词

- ~~最准 / 最全 / 独家~~（俗套且引发质疑）
- ~~检验 / 审查 / 审判~~（过于肃杀，偏离档案调性）
- ~~测试 / 问卷~~（用"测评""评估"更贴公文腔）
- ~~下载 App / 关注公众号~~（我们没有 App / 公众号，别承诺不存在的东西）
