# 答疑口径 Q&A

> 发布后评论区、私信、群聊里可能被问到的问题。每条都准备了一个"可直接复制"的标准回答 + 一个"轻松版"备选。保持口径统一，不要临场发挥出和文档矛盾的话。

---

## 一、关于测评本身

### Q1：这是真的 MBTI 吗？

**标准回答：**
不是。它借用了 MBTI 的"选择题 → 四维度 → 人格类型"这个壳子，但四个维度完全重做了——换成了《人民的名义》剧里真正会吵起来的那四条分野（理想/现实、规则/权变、协作/独断、进取/守成）。所以匹配到的是**剧中 16 位人物**，而不是 INTJ / ENFP 这类抽象标签。

**轻松版：**
不是 MBTI，是"人民的名义版 MBTI"——内核换了，壳没换。

---

### Q2：测出来的结果准不准？

**标准回答：**
它不是严肃的心理学量表，定位是"给剧粉玩的自我投射游戏"。题目和人物映射是基于剧中人物的行为模式做的工程建模，有内部一致性，但并没有经过临床心理学验证。请当娱乐和聊资，别当招聘背调。

**轻松版：**
不准（严肃脸）。但好玩（微笑脸）。

---

### Q3：为什么是 32 题？能不能再短 / 再长一点？

**标准回答：**
32 题是我们在"题量足够覆盖 4 个维度、但又能在 5 分钟内做完"之间权衡出来的数。每个维度 8 题，正反向各半，统计上足够稳定。如果再短，维度判定会抖；再长，完播率会掉。

**轻松版：**
再短不准，再长你会嫌烦。

---

### Q4：我怎么测出的是孙连城 / 刘新建 / 丁义珍？我不服。

**标准回答：**
一种可能：你在 4 个维度上的分数绝对值都比较小，没有明显倾向，算法会落到"没脾气"的角色（典型如孙连城）——这不是说你平庸，只是你在这几条分歧上态度比较中性。另一种可能：某几题你可能按"我希望自己是"而不是"我真实会怎么做"答的，这会带偏结果。可以重测一次，看变化。

**轻松版：**
孙连城不丢人，他就是太佛了。

---

### Q5：为什么没有（我喜欢的某个角色）？

**标准回答：**
我们挑的是 16 位在剧中有足够戏份、且在 4 个维度上位置相对可识别的角色。像陈海、郑西坡、达康夫人、刘慧芬这些，要么戏份不够，要么性格维度和已经选中的角色过于接近。后续如果有反馈集中的角色，会考虑在 v2 里加上。

**轻松版：**
她/他戏份不够，我手下留情了。

---

## 二、关于技术 / 数据

### Q6：我的答题数据会被上传吗？

**标准回答：**
不会。整个站是纯静态的，没有后端、没有数据库、没有账户系统。你的答题结果完全编码在 URL 的 `#` 之后（hash 片段），**浏览器不会把 hash 发送给服务器**。我们既看不到你的测评结果，也不知道你是谁。

**轻松版：**
连我都不知道你测了啥。

---

### Q7：那为什么我复制链接给朋友，他打开看到的是我一样的档案？

**标准回答：**
因为答题结果就编码在那个长长的链接里（`/#/r/v1.xxxx` 这一段）。朋友打开链接时，他的浏览器本地就能根据这段字符算出同一份档案。数据还是没离开过浏览器，只是你**把这串字符主动给他了**。

**轻松版：**
链接自己会带着答案跑。

---

### Q8：能不能下载 App / 关注公众号拿结果？

**标准回答：**
我们没有 App，也没有公众号，更不会索取手机号。就是一个网页，打开即用。

**轻松版：**
没有 App，没有公众号，没有会员，放心。

---

### Q9：支持手机吗？

**标准回答：**
支持。移动端适配做了，iOS / 安卓主流浏览器都能正常跑。分享长图的生成也在手机上测过，可以直接保存到相册。

## 三、关于版权 / 合规

### Q10：这不是在蹭《人民的名义》的热度 / 侵犯版权吗？

**标准回答：**
这是一个**非商业粉丝项目**，不售卖任何商品 / 会员 / 广告位。角色剧照仅用于"让读者识别这个人物是谁"的对话语境下使用。如果剧集出品方（最高人民检察院影视中心 / 天津嘉会文化 / 弘道影业 / 正义网络传媒等）有任何下架要求，我会在接到通知后立刻下架或替换为抽象插画。

**轻松版：**
爱它才做。如果官方说别做，马上下架。

---

### Q11："汉东省"是真的吗？为什么不用"最高人民检察院"之类的名字？

**标准回答：**
汉东省是剧中虚构的省份，不存在现实对应。我们也刻意避开了一切真实的国家机关名称——站点品牌是"汉东省人格档案室"，一个剧内虚构宇宙的虚构机构。这既符合剧的调性，也避免给任何现实机构带来困扰。

**轻松版：**
汉东省是戏里的，站也是戏里的。

---

### Q12：能不能把我的测评结果（尤其是祁同伟、高小琴这类反派）当成真人标签给别人贴？

**标准回答：**
请别这样用。所有人格档案在写作上都刻意保留了**剧中人物的复杂性**——祁同伟虽然走向悲剧，但他的起点、努力、困境都是可理解的；高小琴虽然在反派阵营，但她的果断和清醒也是真的。我们希望它是一次"照见自己"的自我反省，不是"给别人下定义"的武器。

**轻松版：**
测出来是反派？那正好，剧里反派才有戏。

### Q14：开源吗？能不能 Fork 做成别的剧的版本？

**标准回答：**

**轻松版：**
思路免费拿走，代码暂时留我家。

## 五、如果被问到敏感 / 奇怪的问题

以下几类问题**不要展开回答**，用统一的温和短句收束：

| 类型 | 回应 |
| --- | --- |
| 索要剧中真实原型的对应关系（"XX 是现实中的 XX"） | "剧里是虚构的，这个站也就止于剧。" |
| 问是否影射某省 / 某官员 | "剧本身是虚构的，我只做了虚构的延伸。" |
| 拿结果攻击他人（"你看你是祁同伟吧"） | "每个角色都有自己的难处，测评是照镜子不是标签。" |
| 要求把自己的结果"改"成别的 | "这个站没有后端，我也改不了你的结果——重测吧。" |
| 抄袭 / 盗站 | 先礼后兵：邮件沟通 → 平台投诉 → 必要时公开说明。 |
