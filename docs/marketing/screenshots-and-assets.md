## 二、平台适配（剪裁方案）

不同平台对封面比例的要求不同，从源素材裁剪、加工出以下适配图：

### 微博 / 公众号头图（16:9，建议 1200×675）
- **a1**：`01_home_masthead.png` 横向剪到 16:9，副文案大字叠上去"你是《人民的名义》里的哪一号？"
- **a2**：`05_result_top.png` 剪 16:9

### 小红书封面（3:4 或 9:16，建议 1080×1440 或 1080×1920）
- **b1**：`12_mobile_home_full.png` 直接用（430×长自带 9:16 感，需按比例放大到 1080 宽）或裁剪 + 文字海报"终于有人把《人民的名义》做成 MBTI 了"
- **b2**：`09_longimage.png` 本身就是 1080×1920，直接可用

## 四、视频素材（可选，抖音 / 小红书视频）

如果要做 15-30 秒竖版短视频：

1. **0-3s**：首页报头特写 + 绝密章动效
2. **3-6s**：快切 3 张答题页（不同题目，暗示题目丰富）
3. **6-12s**：结果页滚屏展示（人物 → 金句 → 四维条 → 长文）
4. **12-18s**：生成长图动画（点按钮 → canvas 绘制 → 下载）
5. **18-22s**：二维码扫码实拍（展示跨设备复活）
6. **22-30s**：号召 + 域名口播

配乐建议：剧 OST《人民的名义》主题曲 instrumental 版，或者节奏感强的政经类剧配乐（注意版权）。
