# 产品定位

## 一句话定位

**一款《人民的名义》主题的 MBTI 风格人格测评——32 道题，把你匹配到 16 位剧中人物里最像的那一位。**

## 一段话定位

不同于通用 MBTI，我们把坐标系换成了剧里真正吵架的四条政治哲学：**理想 / 现实、规则 / 权变、协作 / 独断、进取 / 守成**。于是你测出的不是 INTJ、ENFP 这类抽象标签，而是沙瑞金、侯亮平、李达康、祁同伟、高育良、高小琴……你在谁的位置上照见自己。全程匿名，无需登录，数据不离开你的浏览器；结果链接可以直接甩给朋友比较，也能一键生成带二维码的竖版长图分享。

## 三条核心卖点（按优先级）

1. **剧粉情感点**：16 位人物都有真·剧照 + 原创人格长文（每人 700-800 字，不是网文水词），金句保留剧里最出圈的那一句。祁同伟那句「**我太想进步了**」原样保留。
2. **不套 MBTI 模板**：四维度完全重做，对应剧里真实存在的政治哲学分野。测出来是高小琴不是 ENTJ，测出来是孙连城不是 ISTP——体验和梗都更准。
3. **档案卷宗风视觉**：整站走"汉东省人格档案室"档案封面美学，印章、宋体、朱砂、牛皮纸、卷宗边角装饰。截图本身就是素材。

## 次级卖点

- **纯静态、零后端**：结果状态全部编码在 URL hash 里，可以直接把链接甩给朋友——打开是同一份档案。
- **分享长图**：1080×1920，带二维码，微信朋友圈 / 小红书 / 微博九图都能用。
- **最近/最远邻居**：结果页除了你本人的档案，还告诉你 16 人里"和你最像的另一个"和"和你最背离的那一个"，制造二次话题。
- **约 5 分钟完成**：32 题，一题一屏，选完即进下一题，不需要耐心。

## 目标受众

**核心盘**（最易转化）
- 《人民的名义》原剧观众（2017 播出时近 8 亿人次观看，沉淀大量长期剧粉）
- MBTI 爱好者、性格测评爱好者
- 爱发"同款人格"到朋友圈的社交传播型用户

**扩散盘**（靠话题带进来）
- 公考 / 体制内 / 政治爱好者
- 国产剧、正剧、主旋律剧讨论社群
- 产品、设计、前端开发者（欣赏静态架构 + 视觉风格）

**低优先**
- 完全没看过剧的用户——能测但缺少情感锚点，转化较低

## 调性 / Tone

- **幽默但不油滑**：剧本身是严肃政治剧，但民间对角色的讨论（祁同伟、李达康、孙连城）有大量梗与自黑，文案可以玩梗但不能低级
- **档案公文感 + 网感并存**：落地页文案用"干部人格档案""被测评人"这种官方腔，朋友圈文案切回口语"测了一下我是沙瑞金哈哈哈"
- **不自夸 / 不吹嘘**：不要说"最准""最全"，剧粉最反感"蹭热度"感。说"一份给剧粉的小玩意儿"

## 避坑清单

- 不使用"最高人民检察院"等真实政府机构名称作为站点品牌
- 不暗示本站与剧集出品方存在合作关系
- 不对外售卖任何商品、会员或广告位（纯粉丝非商业项目）
- 不以测评结果给用户贴负面标签；祁同伟虽然是反派但仍保持可爱自嘲感，避免羞辱
- 角色剧照仅作识别用途，若出品方有正式异议立即下架
